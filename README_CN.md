## 功能特点

 - 支持调节进度.
 - 支持调节速度.
 - 支持设置方向（上下）.
 - 支持设置水波的数量（无上限）.
 - 支持精确定义水波的参数（偏移、拉伸、原始速度）.
 - 支持设置颜色渐变和改变渐变方向.
